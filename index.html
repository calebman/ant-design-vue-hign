AntDesignVueHign

基于ant-design-vue的高阶组件库

更适应业务

从 Stylus 样式过渡到 Less 样式，统一使用预处理器，让 node_modules 不再过度臃肿。

Ant Design 风格

组件库基于 Ant Design 开发，且保证原组件的 API 不受影响。

各种功能优化

新增底部栏目设置，优化行内代码、代码块以及其他各种显示效果和动画，真正实现风格迁移。

# 安装依赖

使用 yarn 安装 AntDesignVueHign 组件：

yarn add ant-design-vue-hign

或者使用 npm 安装它：

npm i ant-design-vue-hign -s

如果你的网络环境不佳，推荐使用 cnpm。

# 使用组件

使用 vue-cli 构建项目，打开 src/main.js 通过合适的方式引入它

注意 AntDesignVueHign 强依赖 AntDesignVue 组件库，使用组件库需要同时引入 AntDesignVue

如果你对 AntDesignVue 的使用不是很了解，请点这里

  1. 全局引入
import Vue from 'vue'
import Antd from 'ant-design-vue'
import AntDesignVueHigh from 'ant-design-vue-hign'
import App from './App.vue'
import 'ant-design-vue/dist/antd.less'
Vue.use(Antd)
Vue.use(AntDesignVueHigh)
new Vue({
  render: h => h(App)
}).$mount('#app')
  1. 按需加载
import { IconPicker } from 'ant-design-vue-hign'
Vue.use(IconPicker)
